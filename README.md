# Camtinel, landing page

Cameroon's offline AI cybersecurity assistant. This is the marketing website.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Building

```bash
npm run build
npm run start
```

## APK

Drop the Android build at `public/camtinel.apk`. The Download button in the hero and download section links to that path.
